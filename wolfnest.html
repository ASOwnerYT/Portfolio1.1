Wolfnest Tenant Portal

Personal Project

Project Overview

Problems: UI is outdated, unwelcoming, and unpleasant to look at.

Proposed Solution: Redesign page using Wolfnest's homepage as a branding guide, and conduct hybrid card sorts to create a more intuitive menu.

Why?

Reasoning

For the past four years I have cringed every month as I paid my rent. The most obvious pain point is the lack of concsistency. There are too many options where I can perform the same task.

The biggest pain point is the interface recall. The same message of "do not use the messages feature" adds to my memory load. The world of UX/UI showed me how I could improve this interface, and so I did.

What?

Research

For any website to be successful, it must have an intuitive navigation. First, I

  • Sorted the navigation options as it seemed intuitive to me
  • Then, I conducted two more hybrid card sorts
  • And finally, compiled the data together to create a new megamenu.

How?

Design

Using trending UI and the Wolfnest Property Management homepage as a brand gruide, I started designing a new tenant portal home page.(View the full Style Tile.)After three iterations, it was time for more testing.

What?

Usability Testing

With 3 usability tests I discovered

  • The footer information was overlooked because users didn't think to scroll
  • Users would rather have the contact information hidden but easily accessible at the top of the page
  • The 2 main CTA buttons were used about 50% of the time

Rinse, & Repeat

Design

As I continued iterating on the designs, I also wanted to make the page mobile friendly.

With multiple levels within the navigation I scrapped my original drop-down and pop-out menu for a nested caret look similar to Amazon. Try the mobile prototype.

Nitty Gritty

Problems I encountered

  • It was difficult to find users for testing. I used what I had: my friends, parents, and children.
  • My original idea for the mobile navigation ended up very clunky. Some of the original component framework was still usable in my new design.
  • Creating the CTA color-change effect was very time consuming and took me several hours. In the future, I hope to have a more skilled mentor to help me with advanced techniques rather than relying on just Youtube videos.
